# Workbench Support

## Minimum Requirements

- macOS 14 Sonoma or later

## Common Install Steps

1. Download the zip from Gumroad.
2. Unzip it.
3. Move `Workbench.app` to `/Applications`.
4. Open `Workbench.app`.

## Troubleshooting

If macOS blocks launch, confirm you downloaded the latest notarized build from Gumroad.

If screenshot or recording tools do not work, open System Settings and check:

- Privacy & Security > Screen Recording
- Privacy & Security > Microphone

If a file operation does not work, check whether the file is locked, read-only, on a disconnected
external drive, or inside a protected system folder.

## Updates

Updates are delivered through Gumroad. Customers can download the newest build from their Gumroad
library after each release.
