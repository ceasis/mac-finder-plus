# Workbench 0.1.0

Initial Gumroad release candidate.

## New

- File browser with list, icon, and multi-pane workflows.
- Inline preview for images, video, and audio.
- Notes with inline images and attachments.
- Snippets for reusable text, images, audio, and files.
- Voice recording with in-app playback.
- Screenshot and screen recording tools.
- Quick filters for recent files, older files, size ranges, files, and folders.
- Media tools for supported image, video, and audio files.
- PDF, document, archive, folder, and file context tools.
- Disk space analyzer with charts and deletion candidates.

## Requirements

- macOS 14 Sonoma or later.
- Optional Screen Recording permission for capture tools.
- Optional Microphone permission for voice recording and narrated capture.

## Notes

This release is prepared for direct download distribution. Public builds should be Developer ID
signed and notarized before upload.
